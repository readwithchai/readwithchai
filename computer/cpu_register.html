CPU Registers

In a CPU, registers are small, high-speed storage locations used to quickly store and retrieve data that the CPU needs to access frequently and perform operations on.

Data Registers:

  • Accumulator (ACC): Used for arithmetic and logic operations.
  • General-Purpose Registers (GPRs): Versatile registers for various purposes during computations.

Address Registers:

  • Memory Address Register (MAR): Holds memory locations' addresses.
  • Memory Buffer Register (MBR) or Memory Data Register (MDR): Temporarily stores data from memory.

Control Registers:

  • Program Counter (PC): Keeps track of the next instruction's memory address.
  • Instruction Register (IR): Holds the current instruction fetched from memory.

These registers play a crucial role in the fetch-decode-execute cycle of the CPU, significantly enhancing its processing capabilities.
